# Simple-Calculator-App-using-Python-Functions

## Project Overview
This project is a simple calculator application built using Python. It allows users to perform basic mathematical operations such as addition, subtraction, multiplication, and division.

## Objectives
- Create separate functions for arithmetic operations
- Accept user input for numbers and operation
- Display the calculated result
- Use built-in functions like `abs()`, `round()`, and `pow()`
- Handle division by zero safely

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- Use of Python built-in functions

## Technologies Used
- Python

## Concepts Used
- Functions
- Conditional statements
- User input
- Error handling
- Built-in functions
